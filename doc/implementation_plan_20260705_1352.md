# 坐标拾取与自动更新工具实现计划

## 背景与目标

为了免去手动测算和修改坐标的麻烦，创建一个辅助小工具。该工具允许用户在 Ginko 窗口上用鼠标选取目标区域，并将获取到的相对坐标（X, Y, W, H）自动更新到 `ahk_example.ahk` 脚本中。

## 实施策略

**[NEW] `scripts/rect_picker.ahk`**

编写一个独立的 AHK v1 脚本，工作流程如下：

1. **触发与交互**：
   - 运行该脚本后，在 Ginko 窗口激活时按 `F8`。
   - 弹出输入框，询问你要更新哪个参数（如输入 `HR`、`PR`、`QTC`）。
   - 提示“请鼠标左键点击区域的【左上角】”。
   - 用户点击后，提示“请鼠标左键点击区域的【右下角】”。
   - 用户再次点击，脚本自动计算出 `W` (宽度) 和 `H` (高度)。

2. **坐标转换**：
   - 获取鼠标点击位置时，强制使用 `CoordMode, Mouse, Window`，确保拿到的是相对于 Ginko 窗口左上角的坐标。

3. **自动更新脚本**：
   - 读取 `ahk_example.ahk` 的内容。
   - 使用正则表达式查找并替换对应的变量赋值行（例如，寻找 `HR_X := \d+` 并替换为 `HR_X := 新坐标`）。
   - 将更新后的内容保存回 `ahk_example.ahk`（保持 UTF-8 BOM 编码）。
   - 弹窗提示更新成功。

## 优势

- **无需复杂的绘图逻辑**：两点点击法（左上、右下）在 AHK 中实现最稳定，不会产生屏幕残留或导致 Ginko 窗口失去焦点。
- **完全自动化**：直接修改目标文件，不需要你手动复制粘贴。
